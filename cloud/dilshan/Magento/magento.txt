	PUT to live
	magento deploy:mode:set production (dont, stuck some times)

	php bin/magento setup:upgrade



**after above, do this**
	php bin/magento setup:static-content:deploy [optional]--area frontend
	php bin/magento setup:static-content:deploy --area adminhtml
Then make permision back to php
	sudo chown -R ftp_admin:www-data ./*   (bitnami daemon instead of www-data)
	sudo chown -R bitnami:daemon ./*
	sudo chmod -R 777 ./*


sudo chown -R ftp_admin:www-data ./
sudo chmod -R 777 ./

**then, if not working** 
	php bin/magento indexer:reindex (**also when slow**)

**enable Module** 
	php bin/magento module:enable mod_name (**Enable Cache**)



php bin/magento maintenance:disable



php bin/magento setup:di:compile

sudo /opt/bitnami/ctlscript.sh restart varnish
